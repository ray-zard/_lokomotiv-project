Песок Строительный

Карьерный (горный) песок — природного происхождения. Неоднороден по составу — содержит примеси глинистых и пылевых частиц, следовательно, имеет более низкую цену на рынке. Применяется для строительных работ нулевого цикла — насыпей, отвалов, подстилающих слоев, оснований дорог. Используется при изготовлении бетона. Благодаря содержанию глины, карьерный песок лучше других видов песка подходит для раствора при кирпичной кладке.

  • Самый универсальный в строительстве вид песка
  • Лучший при кладке кирпича
  • Доступная цена

Минимальный заказ 10т.
